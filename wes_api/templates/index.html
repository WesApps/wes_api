What does it do?:

Wes API scrapes, aggregates, and provides standardized programmatic access to data from sources including Wesleying, Wesleyan Events, Wesleyan Film Series, Usdan, Summerfields, WesMaps, and various others. For more information about what exactly is going on under the hood, check out the about page.
