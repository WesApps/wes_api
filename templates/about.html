About

#WhyWesAPI?

In short, Wesleyan related information is scattered all around the web and can often be very difficult to find and make sense of. Wes API exists to aggregate this data and provide it programmatically to anyone who wishes to use it. This API is also a potential building block for any other Wesleyan-related applications as it abstracts the problem of finding, parsing, and collecting Wesleyan data to one simple API call. Yipee!

#HowDoesItWork?

Wes Application Program Interface is an open source web scraping system that lives on a server and delivers data to clients who make requests for data. The web scraper is collection of scripts to gather data from various sources on the web along with manually inputted data and store it all in a database. When a request comes to the server for information, the server queries the database for relevant data, formats this data, and responds with a message containing the data.

#CanYouAdd...?

Some things are difficult to scrape off the web or simply don't exist online, we we often manually enter them or just live without them. Currently, we don't offer menus for Star & Crescent and Wes Wings. Star & Crescent's information changes every week, making manual entry time consuming, and their menu post on Wesleying has proven tricky to identify and parse (if you're up to the task of writing a scraper for the task, however, let us know!). The menu for Wes Wings is not available online but will be manually entered sometime in the near future.

#ThisDataIsWrong!

Sometimes manual data entry goes etaoin shrdlu and needs to be corrected, so let us know and we'll fix it right away!

#WhoMadeThis?

Wes API was created by Aaron Plave '15, with contributions from Trisha Arora '16, Lili Borland '16, and Matthew Stein '16. The event scraping is based off a WesHappening, a WesHack project. Summerfields and Late Night static data was used with permission from FoodyCall another WesHack project. Film Series static data is provided with permission from the Wesleyan Film Department. Directory information is manually aggregated from all across the web.

If you're interested in contributing in any way, please let us know.
